## 📋 نظرة عامة

نظام إدارة متكامل لشركة **نقاء المتحدة** لتنقية المياه، يشمل إدارة المواعيد، العملاء، المنتجات، المخزون، العهد، المهام، العقود، التقارير المالية، وإدارة المستخدمين.

## ✨ المميزات

### 📊 لوحة التحكم
- إحصائيات حية للمواعيد والإيرادات
- مواعيد اليوم مع تحديثات فورية
- إشعارات ذكية

### 📅 إدارة المواعيد
- جدولة تلقائية للصيانة الدورية (كل 6 أشهر)
- 7 حالات مختلفة (جديد → مكتمل)
- 3 مستويات أولوية (عادي، مستعجل، VIP)
- إسناد فنيين تلقائي حسب المدينة

### 📋 إدارة المهام
- إنشاء وإسناد مهام بين الموظفين
- نظام تعليقات داخل كل مهمة
- تتبع حالة المهام وتنبيهات التأخير

### 📦 إدارة المنتجات
- 15 منتج ببيانات حقيقية من متجر pu.sa
- صور، أوصاف، أسعار، تخفيضات
- فئات (فلاتر تحلية، برادات، بكجات، قطع غيار، ملحقات)

### 🏪 إدارة المخزون
- مخزون منفصل لكل مدينة (22 مدينة سعودية)
- تنبيهات المخزون المنخفض

### 🛡️ إدارة العهد
- نظام طلب/قبول عهدة بين الفنيين والإدارة
- تتبع العهدة لكل فني

### 👥 إدارة العملاء
- سجل كامل لكل عميل
- ربط المواعيد والعقود

### 📄 العقود
- عقود سنوية مع تنبيهات الانتهاء
- تتبع الزيارات المكتملة

### 💰 الحسابات والتقارير
- تقارير الإيرادات حسب المدينة والفني
- طرق دفع متعددة (نقد، تحويل، شبكة)

### 🔐 نظام الصلاحيات
- 5 أدوار (مالك، مدير عام، مدير فرع، خدمة عملاء، فني)
- 14 صلاحية قابلة للتخصيص
- سجل تعديلات شامل (Audit Log)

### 📱 تجاوبية كاملة
- يعمل على الديسكتوب والتابلت والجوال
- شريط تنقل سفلي للجوال
- قائمة جانبية منزلقة (Drawer)

## 🚀 التشغيل

### الطريقة السريعة (بدون تثبيت)
1. افتح [claude.ai](https://claude.ai)
2. ارفع ملف `src-app.jsx` كـ Artifact
3. سيعمل مباشرة في المتصفح

### مع React
```bash
npx create-react-app naqaa-system
cd naqaa-system
cp src-app.jsx src/App.jsx
npm start
```
